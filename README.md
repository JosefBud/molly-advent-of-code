# Advent of Code

## JS

```
// Create directory & boilerplate for a new day's puzzle
$ node newDay.js <year> <day>

// Run a puzzle
$ node index.js <year> <day> -n <puzzle-num>
```
## Python
```
// Create directory & boilerplate for a new day's puzzle
$ node newDay.js <year> <day> -l python

// Run a puzzle
$ node index.js <year> <day> -l python -n <puzzle-num>
```
